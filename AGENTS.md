# Instructions for AI

This repo is a collection of skills and associated settings for Opencode and other agentic harnesses.

## Directories

- automations: AI automations intended to be run standalone or as a batch process
- commands: general AI commands
- droids: Factory/Droid specific agent configs
- skills: general ai skills to be installed

## Languages

Use Python through `uv` (see the @skills/uv-package-manager skill for details)
